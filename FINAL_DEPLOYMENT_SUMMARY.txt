## üêõ Troubleshooting Guide

### Site doesn't load after push?
1. Check GitHub Actions (should show green ‚úÖ)
2. Wait 2-3 minutes (sometimes GitHub is slow)
3. Clear browser cache (Ctrl+Shift+Delete)
4. Check domain DNS (dnschecker.org)

### Images not showing?
1. Check imgur URLs are accessible
2. Open DevTools ‚Üí Network tab
3. Verify image quality parameters
4. Try incognito/private mode

### Build fails in Actions?
1. Check Actions tab for error message
2. Verify npm install works locally
3. Ensure all files are committed
4. Check for syntax errors

### CSS/Styling wrong?
1. Clear browser cache completely
2. Check CSS file loaded (network tab)
3. Verify theme toggle working
4. Inspect element for errors
